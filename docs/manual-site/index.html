AutoCoder マニュアル

自律型AIコーディング＆コンテンツ作成エージェントシステム

AutoCoderとは？

AutoCoderは、長時間実行可能な自律型AIコーディングおよびコンテンツ作成エージェントシステムです。最小限の人間の介入で、複数のセッションにわたって完全なアプリケーションを構築したり、コンテンツを作成したりできます。

🤖

2エージェントパターン

初期化エージェントとコーディングエージェントの連携により、マルチセッション開発を実現

📊

リアルタイムUI

React製のカンバンボードで進捗をリアルタイムに可視化

🔄

自動継続

セッション間で進捗を永続化し、中断したところから自動再開

🎯

スキルベースルーティング

タスクに最適なエージェントを自動選択

🛡️

3層セキュリティ

OSサンドボックス、ファイル制限、コマンドホワイトリスト

⚡

YOLOモード

テストをスキップした高速プロトタイピング

クイックスタート

ターミナル1: UIを起動 
cd /Users/hiroyuki/autocoder2/autocoder
./start_ui.sh
# http://localhost:5173 でカンバンボードが開きます
ターミナル2: タスクを実行 
cd /Users/hiroyuki/autocoder2/autocoder/generations/my-project
claude
/task Reactでtodoアプリを作成
